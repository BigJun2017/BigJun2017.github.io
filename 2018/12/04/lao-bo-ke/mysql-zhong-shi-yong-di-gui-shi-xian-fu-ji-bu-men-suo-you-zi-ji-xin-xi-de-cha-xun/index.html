2、创建存储过程.
执行下面的语句就可以创建function了，也就是执行语句创建存储过程。

create function getChildrenOrgOfHr(orgid INT)
returns varchar(4000)
BEGIN
DECLARE oTemp VARCHAR(4000);
DECLARE oTempChild VARCHAR(4000);

SET oTemp = '';
SET oTempChild = CAST(orgid AS CHAR);

WHILE oTempChild IS NOT NULL
DO
SET oTemp = CONCAT(oTemp,',',oTempChild);
SELECT GROUP_CONCAT(deptId) INTO oTempChild FROM hr_dept WHERE FIND_IN_SET(parent_id,oTempChild) > 0;
END WHILE;
RETURN oTemp;
END

3、测试

select * from  hr_dept where FIND_IN_SET(deptId,getChildrenOrgOfHr(2));

4、原理
其实就是一个递归的过程，在存储过程中最终返回的是一个字符串数组，这个字符串数组中包含当前父级节点和所有子级节点的id,而这个字符串数组是通过递归进行拼接逐次增加的哦。
5、可能会出现的问题：

Data too long for column 'oTemp' at row 1
